# degree-roadmap-app
# Degree Roadmap & Professor Recommendation Platform

## Project Description

This project is a social-based web application designed to help university students plan their academic path toward graduation. The system generates a personalized course roadmap based on the classes a student has already completed and the degree requirements remaining.

In addition to academic planning, the application provides professor recommendations using aggregated reviews from students and external rating sources such as Rate My Professor. Users will also be able to contribute their own reviews and ratings to help other students choose professors more effectively.

The goal of the application is to simplify degree planning, improve course selection decisions, and create a community-driven resource for academic guidance.
